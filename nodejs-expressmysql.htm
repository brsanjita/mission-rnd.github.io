NodeJS Express with MySQL.

Overview

The objective of this activity is to understand how databases are integrated to a Express project.

Install mysql node module using npm install mysql to connect to the mysql server installed in your local machine.

Project Activities

ExpressMysql

You will implement the following components to pass all the test cases.

  • Setup express server
  • Setup Route to the students resources such that requests with url path "/students/" are routed to routes/students.js file
  • Implement CRUD operations on Students object on route /students/. Use REST recommended URL specifications.
  • Use MySQL database for persistent storage of the data.
