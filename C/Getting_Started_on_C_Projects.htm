Pre Requisites ,Complete these by September 14

  1. 1)Install Git from http://git-scm.com/downloads 
  2. 2)Allow the installer to add git.exe to your PATH environment variable.(Select second option while installing )
  3. 3)To configure the paths manually, add <InstallationDir>/Git/bin , and <InstallationDir>/Git/cmd to the PATH environment variable.
  4. 4)[SSH Path Setup]Add the following path to Environment variables <InstallationDir>/Git/usr/bin .Installation Dir is where you installed git .Ex path :C:\Program Files\Git\usr\bin
  5. 5] After installation is successful open command prompt and type "git". You should see git usage commands.If any error message comes you need to Add git to the path .
  6. 6)Type "ssh" to check whether its configured properly or not ,or else add the ssh path .
  7. 7)Signup for a github account at: http://github.com/ if you dont have a github account
  8. 8)Install Visual Studio Express 2013 for Windows Desktop. Installer Links are Below .

Getting Started with the Tool [Read the Following After you got the Tool ,There is No use Reading it now ]

  1. For every launch, github username and password have to be provided to the tool configuration settings.

  2. Click on Configure Git and Github ,to configure properly (Make sure ,ssh is added in ENV_PATH)
  3. Following launch screen will appear once the tool is configured with required github credentials.

  4. Click on create new project and select the project that you want to start working with

  5. The project home screen shows the activities that are completed and the activities that haven't been completed. It has the options to test and submit your code.

  6. Open the cloned project in Microsoft visual studio 2013 and start completing activities in the required order.
  7. You can run All tests from Visual studio itself ,From Test->Run->AllTests .You can view the Test cases which failed here.Once you passed all test cases for a particular question or all testcases ,You can start Test and submit from the tool
  8. Complete the assignments by coding the activities in the project directory. Each project is broken into a set of activities. Once you have coded an activity, you can verify if your code is correct by running predefined test cases. The pre-defined tests are under the spec folder.

Steps to run Tests in visual studio , Follow these steps after you installed Project Tracker Tool .

  • Open Visual studio 2013 after installation is successful
  • Select FILE --> Open Project. Choose .sln file of the project you just donwloaded from tool .
  • You will see spec and src projects. spec project contains all test files and src project contains all source files.
  • Now run tests from TEST --> Run --> All Tests. You will see test results on left pane.
  • If you want to debug any test, keep a breakpoint in the file you want to debug and run test from TEST --> Debug --> All Tests.
