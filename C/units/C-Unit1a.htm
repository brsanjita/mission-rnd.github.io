Structure:

The stucture looks similar to the unit-1. The only change is similar to the excercies, you will have assignments in this solution.

You need do these steps.
- Complete the assignments.
- Run all the tests. Make them pass.
- Make all tests pass in Project Tracker tool.
- Submit from Project Tracker tool.

Main.cpp
Assignment_01.cpp, Assignment_02.cpp, Assignment_03.cpp // Assignments
