C-Unit0

Goal of Unit-0 is to make the flow work end to end.

All the steps:

- Open the Visual Studio solution file (ProjectName.sln file).
- Implement the function 'Sum'.
- Run all the tests.
- Test and Submit from this Project Tracker tool.

Each Project will have a tutorial like this one, which will give a breif overview of the Tasks you will be doing and sample codes.

Open the solution file:

Go to the directory where you saved and open the ProjectName.Sln file using Visual Studio 2013 express. You will see 2 projects, src and spec. Src project has the files which you need to code.

Implement the function 'Sum':

Write your code in the 'Sum' function stub in the files present in src->Source folder To pass sample inputs to your 'Sum' function and observe the output, Call the function from Main file which has a main function. It is recommended to test your functions only from here.

Run all the tests:

After you code your function stub, Run Tests from Test->Run->All Tests. It runs testcases of All problems in the project. It shows the list of test cases passed or failed, You can click on failed test cases and see in bottom left window where they failed or the Expected result.

Test and Submit from Project Tracker tool:

Once a problem has passed all tests related to it, go to the Project Tracker Tool, and then click on The Problem you just coded. (If the code you wrote isn't visible in the right window, do click on any other problem and click back on this problem), Now click on Test on the above. It takes some time for first time run. You will see Test Failed or Tests Passed Message. If failed go to VS2013 and code again correctly. Once all tests have been cleared, Click on Submit.

Unable to submit?

The purpose of this unit is find the problems in the end to end process of getting new units and submitting you code. It's common to hit problems in the process. Check the forum for solutions.
