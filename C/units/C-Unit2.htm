C-Unit2

Goal of Unit-2 is to understand
- functions with variable number of arguments
- passing functions as arguments to other functions.
- type casting and type definition
- strings and memory

Structure:

The structure of the unit is it contains some lessons and exercises.
In lessons, you will update the test cases (spec) with expected output.
In exercises, you will update the function stubs (src) with functionality.
You need do these steps.
- Open the Visual Studio solution file (ProjectName.sln file).
- Complete the lessons and excerices.
- Run all the tests.
- Test and Submit from this Project Tracker tool.

Read comments with complete focus

Instruction are given as comments in every file. Read them carefully in these files:
Main.cpp
Lessons in files with prefix 'L0'
Spec files for lessons with prefix 'L0' and suffix 'Spec.cpp'
Excercises in files with prefix 'E0'

The comments contians
- Instruction about how to complete this unit.
- Study links to read and watch related concepts, don't miss them.
